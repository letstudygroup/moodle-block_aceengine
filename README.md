# ACE Dashboard Block for Moodle

A Moodle block plugin that displays student engagement scores, mastery levels, and learning analytics from the [ACE (Adaptive Challenge Engine)](https://plugins.letstudy.gr/moodle-local_ace) local plugin.

## Features

- Displays engagement and mastery scores for the current student in a course.
- Visual progress indicators with color-coded score levels.
- Links to the full ACE analytics dashboard.
- Multi-language support (English and Greek).
- Privacy compliant with full GDPR provider.

## Requirements

- Moodle 4.5 or later
- PHP 8.1 or later
- [local_ace](https://plugins.letstudy.gr/moodle-local_ace) plugin installed and configured

## Installation

1. Download the plugin and extract it to `/blocks/ace/` in your Moodle installation.
2. Visit **Site administration > Notifications** to complete the installation.
3. Add the block to any course page via the block drawer.

## Usage

1. Navigate to a course where ACE is enabled.
2. Turn editing on and add the **ACE Dashboard** block.
3. Students will see their engagement and mastery scores.
4. Teachers and managers will see aggregated course analytics.
